# Intelligent Purchase Timing and Price Decision System

A smart web-based system that tracks product prices and intelligently recommends optimal purchase timing.

## Features

- ✅ Price tracking across e-commerce platforms
- ✅ ML-based price trend prediction
- ✅ Price surge detection
- ✅ User personalization
- ✅ Seller strategy analysis
- ✅ Fake discount detection
- ✅ Smart purchase recommendations
- ✅ Email notifications
- ✅ Dark mode support
- ✅ Fully responsive design

## Tech Stack

- **Frontend**: React.js
- **Backend**: Flask (Python)
- **Database**: SQLite
- **ML**: scikit-learn, pandas, numpy
- **Charts**: Recharts

## Installation

```bash
